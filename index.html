Analysing Social Data

  • This project finds sub-communities/clusters in a larger community. Detatiled explanation of community finding can be found Here
  • Its based on Graph Data Structure. Nodes represent individuals in the community where as Edges represent relations/connections between individuals in the community
  • Girvan-Newman Edge Betweenness Algorithm is used to find communities whereas Brande's Algorithm is used to calculate Edge Betweenness
  • Girvan-Newman Modularity method is used to calculate Modularity
